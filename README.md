## Setup Instructions

### TechnicalTestEngaged (Laravel)
1. `cd TechnicalTestEngaged`
2. `composer install`
3. Copy `.env.example` to `.env` 
4. `php artisan key:generate`
5. `php artisan serve`

### TechnicalTestEngagedFront (Vue.js)
1. `cd frontend`
2. `npm install`
3. Copy `.env.example` to `.env` 
4. `npm run serve`
